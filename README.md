## 🗺️ How to Study

1. Start with `00-fundamentals` — core concepts all roles share
2. Pick your target cloud (`01-azure` / `02-aws` / `03-gcp`)
3. Read `04-multi-cloud-comparison` to understand service equivalents
4. Study `05-app-architectures` — interviewers LOVE architecture questions
5. Go deep on your role-specific folder (`06`, `07`, or `08`)
6. Revise `09-interview-qa` the night before your interview
